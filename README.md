# 🎮 Tkinter Tic Tac Toe (2-Player)

A simple 2-player **Tic Tac Toe game** made with **Python** and **Tkinter**. Two players take turns clicking buttons to place their symbol (`X` or `O`) on a 3x3 grid. The game automatically checks for wins or draws and displays the result.

## 🖥️ Features

- 2-player mode (local)
- Clear GUI using Tkinter
- Automatically detects wins and draws
- Turn indicator (shows who's next)

## 🛠 Requirements

- Python 3.x
- Tkinter (comes pre-installed with most Python versions)
